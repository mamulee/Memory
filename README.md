# Memory

[메모리 SNS 웹 사이트 바로가기](https://memory-webapp.vercel.app/)

### [ 프로젝트 소개 ]

일상을 기억하며 **추억**을 공유하는 SNS 서비스

- 기간 : 2023.08.28 ~ 2023.10.13
- 참여 인원 : 백엔드 1명, 프론트엔드 1명
- 목표
  - 프로젝트 기획, 설계, 개발, 배포, 유지 보수까지 전 과정을 경험
  - 프로젝트를 통해 협업 능력 향상
  - 프로젝트를 통해 문제 해결 능력 향상

### [ 주요 기능 ]

- 회원가입
- 로그인
- 게시글 작성 / 수정 / 삭제
- 댓글 작성 / 수정 / 삭제
- 게시글 저장 / 저장 취소
- 게시글 반응 (좋아요 / 슬퍼요 / 화나요)
- 회원 팔로우 / 팔로잉
